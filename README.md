# Student-Grade-Calculator-using-R-Shiny
This is simple student grade calculator Shiny App.  We can enter student marks of three subject (we can increase subject if we wish so...) and then Shiny App show their grade and bar charts. Here I learned skeleton of every Shiny App, Shiny package working, UI, Server, FLuidPage, Sliderinput, Sidebarlayout, renderprint, wellpanel, sidebarpanel etc.
